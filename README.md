# Cloud Security Capstone

## Project Title

Design and Implementation of a Secure Cloud Infrastructure for Cyberattack Detection and Automated Prevention Using Google Cloud Platform

---

## Objectives

- Build a secure cloud infrastructure
- Deploy a web application
- Simulate cyberattacks
- Detect attacks through logging and monitoring
- Implement preventive security controls
- Automate response to selected attacks

---

## Technologies

- Google Cloud Platform
- Ubuntu Server
- Flask
- Python
- Git & GitHub
- Cloud Logging
- Cloud Monitoring
- VirtualBox
- Kali Linux
- Nmap
- Hydra
- Gobuster
- Nikto
